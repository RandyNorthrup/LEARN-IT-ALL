## Running the Quality Checker

```bash
# Check all courses
python3 scripts/course-quality-check.py

# Check a specific course
python3 scripts/course-quality-check.py --course python-basics

# Verbose output
python3 scripts/course-quality-check.py --verbose
```

The checker validates:
- **Structure** — course.json exists, has required fields, ID matches directory
- **Completeness** — all listed lessons exist, no orphans, no duplicates, sequential chapter ordering
- **Quality** — valid frontmatter, non-empty lesson content
- **Exercises** — JSON validity, required fields (id, title)
- **Quizzes** — JSON validity, required fields (id, title, questions)
- **Accuracy** — difficulty level, language, tags, estimated hours

Exit code: `0` = all pass, `1` = failures found.
