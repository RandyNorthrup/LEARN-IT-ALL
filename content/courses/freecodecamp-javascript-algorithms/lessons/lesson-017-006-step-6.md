# Step 6

You will need a function to generate a range of numbers.

Declare an empty `range` function which takes a `start` and `end` parameter. Use the `Array()` constructor and implicitly return an empty array.

## Hints

1. You should declare a `range` variable.
2. You should use `const` to declare your `range` variable.
3. Your `range` variable should be a function.
4. Your `range` function should use arrow syntax.
5. Your `range` function should take a `start` parameter first.
6. Your `range` function should take an `end` parameter second.
7. Your `range` function should use an implicit return. Remember that this means you will not use curly brackets.
8. Your `range` function should use the `Array()` constructor. Primitive constructors do not need the `new` keyword.
9. You should not pass anything to the `Array()` constructor.
