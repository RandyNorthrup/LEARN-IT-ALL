# Step 19

Append the `input` element to your `container` element as a child.

You should now be able to see the cells of your spreadsheet.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <link rel="stylesheet" href="./styles.css" />
    <title>Functional Programming Spreadsheet</title>
  </head>
  <body>
    <div id="container">
      <div></div>
    </div>
    <script src="./script.js"></script>
  </body>
</html>
```

## Hints

1. You should call the `.appendChild()` method on your `container` element.
2. You should pass your `input` element to the `.appendChild()` method.
