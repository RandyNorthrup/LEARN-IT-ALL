# Step 8

The same syntax can be used to change the value of an existing key.

Just before the `print()` call, access the `'species'` key and reassign its value to `'Cavia porcellus'`.

## Starter Code

```html
--fcc-editable-region--
copper = {
    'species': 'guinea pig',
    'age': 2
}
copper['food'] = 'hay'

print(copper)
--fcc-editable-region--
```

## Hints

1. You should reassign the `copper['species']` to the string `'Cavia porcellus'` before the `print()` call.
