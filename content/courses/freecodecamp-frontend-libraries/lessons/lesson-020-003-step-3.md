# Step 3

Below the `p` element, create a `div` element with the class `btn-container`.

Inside the `div` element, create a `button` element.

This `button` should have an `id` of `happy-btn`, a class of `emoji-btn` and an `aria-label` of `Happy face emoji`.

## Hints

1. Create a new `div` element below the `p` element.
2. The new `div` element should have a class of `btn-container`.
3. There should be a `button` element inside the `div` element.
4. The `button` element should have an `id` of `happy-btn`.
5. The `button` element should have a `class` of `emoji-btn`.
