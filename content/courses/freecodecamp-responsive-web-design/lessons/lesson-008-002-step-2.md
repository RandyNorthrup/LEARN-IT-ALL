# Step 2

Nest a `head` element within the `html` element. Just after the `head` element, add a `body` element.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
--fcc-editable-region--

--fcc-editable-region--
</html>
```

## Hints

1. You should have an opening `head` tag.
2. You should have a closing `head` tag.
3. You should have an opening `body` tag.
4. You should have a closing `body` tag.
5. There should only be one `head` opening tag.
6. There should only be one `head` closing tag.
7. There should only be one `body` opening tag.
8. There should only be one `body` closing tag.
9. Your `head` element should be empty.
10. Your `body` element should be empty.
11. Your `body` element should be placed after your `head` element.
12. Your `head` element should be within the `html` element.
13. Your `body` element should be within the `html` element.
