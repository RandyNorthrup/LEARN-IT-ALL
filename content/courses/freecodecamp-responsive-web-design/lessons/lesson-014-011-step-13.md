# Step 13

As the form will have three distinct sections, add three `fieldset` elements within the `form` element.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Registration Form</title>
    <link rel="stylesheet" href="styles.css" />
  </head>
--fcc-editable-region--
  <body>
    <h1>Registration Form</h1>
    <p>Please fill out this form with the required information</p>
    <form method="post" action='https://register-demo.freecodecamp.org'>
    
    </form>
  </body>
--fcc-editable-region--
</html>
```

## Hints

1. You should add three `fieldset` elements.
2. The three `fieldset` elements should be within the `form`.
3. The three `fieldset` elements should be siblings.
