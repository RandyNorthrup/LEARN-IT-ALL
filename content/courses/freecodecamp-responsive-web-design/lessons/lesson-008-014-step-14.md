# Step 14

Now that you've got one marker centered with color, it's time to add the other markers.

In the `container` `div`, add two more `div` elements and give them each a class of `marker`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Colored Markers</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>CSS Color Markers</h1>
--fcc-editable-region--
    <div class="container">
      <div class="marker">
      </div>
    </div>
--fcc-editable-region--
  </body>
</html>
```

## Hints

1. Your first new `div` element should have an opening tag.
2. Your first new `div` element should have a closing tag.
3. Your second new `div` element should have an opening tag.
4. Your second new `div` element should have a closing tag.
5. Your new `div` elements should be within the `div` with the class `container`.
6. Your new `div` elements should both have a class of `marker`.
