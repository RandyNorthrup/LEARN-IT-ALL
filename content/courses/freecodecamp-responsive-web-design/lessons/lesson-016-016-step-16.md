# Step 16

Time to style the keys themselves. Create a `class` selector for the `.key` elements. Set the `background-color` set to the value `#ffffff`, the `position` property to `relative`, the `width` property to `41px`, and the `height` property to `175px`.

    <meta charset="UTF-8" />

    <link rel="stylesheet" href="./styles.css">
  </head>
  <body>
    <div id="piano">
      <div class="keys">

## Hints

1. You should have a `.key` selector.
2. Your `.key` selector should have a `background-color` property set to `#ffffff`.
3. Your `.key` selector should have the `position` property set to `relative`.
4. Your `.key` selector should have a `width` property set to `41px`.
5. Your `.key` selector should have a `height` property set to `175px`.
