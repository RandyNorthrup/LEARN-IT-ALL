# Step 4

The `charset` attribute specifies the character encoding used by the document. `utf-8` (Unicode Transformation Format â€“ 8-bit) is a character encoding standard used for electronic communication.

Inside the `head` element, nest a `meta` element with the attribute `charset` set to `"utf-8"`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
--fcc-editable-region--
  <head>
    <title>Colored Markers</title>
  </head>
--fcc-editable-region--
  <body>
  </body>
</html>
```

## Hints

1. You should have one `meta` element.
2. Your `meta` element should be a void element, it should not have a closing tag `</meta>`.
3. Your `meta` element should have a `charset` attribute set to the value `"utf-8"`.
4. Your `meta` element should be inside the `head` element.
