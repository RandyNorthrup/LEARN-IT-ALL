# Step 9

Now, get rid of the horizontal scroll-bar, by setting the `body` default `margin` added by some browsers to `0`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Registration Form</title>
    <link rel="stylesheet" href="styles.css" />
  </head>
  <body>
    <h1>Registration Form</h1>
    <p>Please fill out this form with the required information</p>
  </body>
</html>
```

## Hints

1. You should add `margin` within the `body` element selector.
2. You should give the `margin` a value of `0`.
