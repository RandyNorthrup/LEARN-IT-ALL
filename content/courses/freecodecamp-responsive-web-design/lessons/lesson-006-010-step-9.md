# Step 9

Create an `h2` element in the `section` element and give it the text `Coffee`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Cafe Menu</title>
  </head>
  <body>
    <main>
      <h1>CAMPER CAFE</h1>
      <p>Est. 2020</p>
--fcc-editable-region--
      <section>
      </section>
--fcc-editable-region--
    </main>
  </body>
</html>
```

## Hints

1. You should have an opening `<h2>` tag.
2. You should have a closing `</h2>` tag.
3. You should not change your existing `section` element. Make sure you did not delete the closing tag.
4. Your `h2` element should be within your `section` element.
5. Your `h2` element should have the text `Coffee`.
