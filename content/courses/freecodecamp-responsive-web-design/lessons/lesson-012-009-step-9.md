# Step 9

Give your `.background-buildings` element a `width` and `height` of `100%` to make it the full width and height of its parent, the `body`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">    
  <head>
    <meta charset="UTF-8">
    <title>City Skyline</title>
    <link href="styles.css" rel="stylesheet" />
  </head>

  <body>
    <div class="background-buildings"></div>
  </body>
</html>
```

## Hints

1. You should use the `background-buildings` class to select the correct element.
2. Your `.background-buildings` element should have a `width` of `100%`.
3. Your `.background-buildings` element should have a `height` of `100%`.
