# Step 2

Add two `meta` tags, one to optimize your page for mobile devices, and one to specify an accepted `charset` for the page.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Piano</title>
    --fcc-editable-region--

    --fcc-editable-region--
  </head>
  <body></body>
</html>
```

## Hints

1. You should add two `meta` elements to your page.
2. You should have two `meta` elements.
3. One `meta` element should have a `name` set to `viewport`, and `content` set to `width=device-width, initial-scale=1.0`.
4. The other `meta` element should have the `charset` attribute set to `UTF-8`.
