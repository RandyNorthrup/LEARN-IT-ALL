# Step 11

Now target your `#piano` element with an `id` selector. Set `background-color` property to `#00471b`, the `width` property to `992px` and the `height` property to `290px`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Piano</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="./styles.css">
  </head>
  <body>
    <div id="piano">
      <div class="keys">
        <div class="key"></div>
        <div class="key black--key"></div>
        <div class="key black--key"></div>
        <div class="key"></div>
```

## Hints

1. You should have a `#piano` selector.
2. Your `#piano` selector should have the `background-color` property set to `#00471b`.
3. Your `#piano` selector should have a `width` property set to `992px`.
4. Your `#piano` selector should have the `height` set to `290px`.
