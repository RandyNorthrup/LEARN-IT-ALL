# Step 10

Nest a `div` with a class of `bb1` in the background buildings container. Open your `styles.css` file, and give `.bb1` a `width` of `10%` and `height` of `70%`. "bb" stands for "background building", this will be your first building.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">    
  <head>
    <meta charset="UTF-8">
    <title>City Skyline</title>
    <link href="styles.css" rel="stylesheet" />
  </head>

  <body>
--fcc-editable-region--
    <div class="background-buildings"></div>
--fcc-editable-region--
  </body>
</html>
```

## Hints

1. You should create a new `div` element.
2. You should give the new `div` a class of `bb1`.
3. You should use a `.bb1` class selector to style the element.
4. You should give the `.bb1` element a `width` of `10%`.
5. You should give the `.bb1` element a `height` of `70%`.
