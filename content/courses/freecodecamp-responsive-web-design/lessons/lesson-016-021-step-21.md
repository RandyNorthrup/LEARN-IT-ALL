# Step 21

The piano needs the freeCodeCamp logo to make it official.

Add an `img` element before your `.keys` element. Give the `img` a `class` of `logo`, and set the `src` to `https://cdn.freecodecamp.org/platform/universal/fcc_primary.svg`. Give it an `alt` text of `freeCodeCamp Logo`.

## Hints

1. You should add a new `img` element.
2. Your `img` element should come before your first `.keys` element.
3. Your `img` element should have a `class` set to `logo`.
4. Your `img` element should have a `src` set to `https://cdn.freecodecamp.org/platform/universal/fcc_primary.svg`.
5. Your `img` element should have an `alt` attribute set to `freeCodeCamp Logo`.
6. Remember that casing and spelling matter.
