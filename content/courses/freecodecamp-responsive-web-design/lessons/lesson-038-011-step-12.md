# Step 12

To enable navigation on the page, add an unordered list with the following three list items:

- `INFO`
- `HTML`
- `CSS`

The list items text should be wrapped in anchor tags.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta name="description" content="freeCodeCamp Accessibility Quiz practice project" />
    <title>Accessibility Quiz</title>
    <link rel="stylesheet" href="styles.css" />
  </head>
  <body>
    <header>
      <img id="logo" alt="freeCodeCamp" src="https://cdn.freecodecamp.org/platform/universal/fcc_primary.svg">
      <h1>HTML/CSS Quiz</h1>
--fcc-editable-region--
      <nav>

      </nav>
--fcc-editable-region--
    </header>
    <main></main>
  </body>
</html>
```

## Hints

1. You should nest one `ul` element inside the `nav`.
2. You should nest three `li` elements inside the `ul` element.
3. You should nest one `a` element inside each `li` element.
4. You should give the first `a` element the text `INFO`.
5. You should give the second `a` element the text `HTML`.
6. You should give the third `a` element the text `CSS`.
