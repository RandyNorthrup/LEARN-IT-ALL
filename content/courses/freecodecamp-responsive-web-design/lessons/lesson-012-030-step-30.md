# Step 30

Nest six `div` elements within `.foreground-buildings` and give them the classes of `fb1` through `fb6` in that order. "fb" stands for "foreground building". These will be six more buildings for the foreground.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">    
  <head>
    <meta charset="UTF-8">
    <title>City Skyline</title>
    <link href="styles.css" rel="stylesheet" />   
  </head>

  <body>
    <div class="background-buildings">
      <div></div>
      <div></div>
      <div class="bb1">
        <div class="bb1a"></div>
        <div class="bb1b"></div>
        <div class="bb1c"></div>
        <div class="bb1d"></div>
      </div>
      <div class="bb2"></div>
      <div class="bb3"></div>
      <div></div>
      <div class="bb4"></div>
      <div></div>
      <div></div>
    </div>
--fcc-editable-region--
    <div class="foreground-buildings"></div>
--fcc-editable-region--
  </body>
</html>
```

## Hints

1. You should create a new `div` with a class of `fb1`.
2. You should create a new `div` with a class of `fb2`.
3. You should create a new `div` with a class of `fb3`.
4. You should create a new `div` with a class of `fb4`.
5. You should create a new `div` with a class of `fb5`.
6. You should create a new `div` with a class of `fb6`.
7. You should place these new `div` elements within the `.foreground-buildings` element.
8. You should place the new `div` elements in the correct order.
