# Step 3

Normalise your page's sizing, by removing the `body` element's `margin` and `padding`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  
  <head>
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="./styles.css" />
    <title>Penguin</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>

  <body>
  </body>
</html>
```

## Hints

1. You should give `body` a `margin` of `0`.
2. You should give `body` a `padding` of `0`.
