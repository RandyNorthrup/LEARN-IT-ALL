# Step 5

Within your `.keys` element, add seven `div` elements. Give them all the class `key`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Piano</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>
  <body>
    --fcc-editable-region--
    <div id="piano">
      <div class="keys"></div>
    </div>
    --fcc-editable-region--
  </body>
</html>
```

## Hints

1. You should create seven new `div` elements.
2. Your seven new `div` elements should be within your `.keys` element.
3. Your seven new `div` elements should all have the `class` set to `key`.
