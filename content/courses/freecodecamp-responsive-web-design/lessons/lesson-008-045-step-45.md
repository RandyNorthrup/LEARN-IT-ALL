# Step 45

Update the CSS class selector `.two` so it targets the new `green` class. And update the `.three` class selector so it targets the new `blue` class.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Colored Markers</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>CSS Color Markers</h1>
    <div class="container">
      <div class="marker red">
      </div>
      <div class="marker green">
      </div>
      <div class="marker blue">
      </div>
    </div>
  </body>
</html>
```

## Hints

1. Your code should no longer have a `.two` class selector.
2. You should use a class selector to target the class `green`.
3. Your `.green` CSS rule should have a `background-color` property set to `rgb(0, 0, 0)`.
4. Your code should no longer have a `.three` class selector.
5. You should use a class selector to target the class `blue`.
6. Your `.blue` CSS rule should have a `background-color` property set to `rgb(0, 0, 0)`.
