# Step 10

Above the `.ground` element, add a `div` with a `class` of `penguin`. This `div` will contain Flappy Penguin.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  
  <head>
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="./styles.css" />
    <title>Penguin</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>

--fcc-editable-region--
  <body>

    <div class="ground"></div>
  </body>
--fcc-editable-region--
</html>
```

## Hints

1. You should add a new `div` within the `body`.
2. You should give the `div` a `class` of `penguin`.
3. You should place `div.penguin` before `div.ground`.
