# Step 4

Normalise your page, by setting the `width` to `100%`, and `height` to `100vh`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  
  <head>
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="./styles.css" />
    <title>Penguin</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  </head>

  <body>
  </body>
</html>
```

## Hints

1. You should give `body` a `width` of `--fcc-expected--`, but found `--fcc-actual--`.
2. You should give `body` a `height` of `--fcc-expected--`, but found `--fcc-actual--`.
