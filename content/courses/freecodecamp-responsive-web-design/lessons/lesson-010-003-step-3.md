# Step 3

Within your `body`, create a `main` element. Then in that element, create a `section` with a `class` set to `heading`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Magazine</title>
    <link
      href="https://fonts.googleapis.com/css?family=Anton%7CBaskervville%7CRaleway&display=swap"
      rel="stylesheet"
    />
    <link
      rel="stylesheet"
      href="https://use.fontawesome.com/releases/v5.8.2/css/all.css"
    />
    <link rel="stylesheet" href="styles.css" />
  </head>
  <body>
--fcc-editable-region--

--fcc-editable-region--
  </body>
</html>
```

## Hints

1. You should have a `main` element.
2. Your `main` element should be within your `body` element.
3. You should have a `section` element.
4. Your `section` element should be within your `main` element.
5. Your `section` element should have the `class` set to `heading`.
