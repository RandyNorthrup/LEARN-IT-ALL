# Step 55

## Hints

1. You should remove the `padding-left` property.
2. You should remove the `padding-right` property.
3. You should remove the `padding-top` property.
4. You should remove the `padding-bottom` property.
5. You should set the `padding` property to `20px`.
6. Your `.menu` element should have a `padding` value of `20px`.
