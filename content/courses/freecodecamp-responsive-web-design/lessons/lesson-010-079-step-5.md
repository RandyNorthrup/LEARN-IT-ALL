# Step 5

After your `img` element, add an `h1` element with the `class` set to `hero-title` and the text set to `OUR NEW CURRICULUM`, followed by a `p` element with the `class` set to `hero-subtitle` and the text set to `Our efforts to restructure our curriculum with a more project-based focus`.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />

    <link

--fcc-editable-region--
    <main>
      <section class="heading">
        <header class="hero">
          <img
            src="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"

        </header>
```

## Hints

1. You should create an `h1` element.
2. Your `h1` element should come after your `img` element.
3. Your `h1` element should have the `class` set to `hero-title`.
4. Your `h1` element should have the text set to `OUR NEW CURRICULUM`.
5. You should create a new `p` element.
6. Your `p` element should come after your `h1` element.
7. Your `p` element should have the `class` set to `hero-subtitle`.
8. Your `p` element should have the text set to `Our efforts to restructure our curriculum with a more project-based focus`.
