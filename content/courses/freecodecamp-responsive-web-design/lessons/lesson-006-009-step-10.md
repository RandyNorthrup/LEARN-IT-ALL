# Step 10

Until now, you've had limited control over the presentation and appearance of your content. To change that, add a `style` element within the `head` element.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
--fcc-editable-region--
  <head>
    <meta charset="utf-8" />
    <title>Cafe Menu</title>
  </head>
  <body>
    <main>
      <h1>CAMPER CAFE</h1>
      <p>Est. 2020</p>
      <section>
        <h2>Coffee</h2>
      </section>
    </main>
  </body>
--fcc-editable-region--
</html>
```

## Hints

1. Your code should have an opening `<style>` tag.
2. Your code should have a closing `</style>` tag.
3. Your `style` element should be nested in your `head` element.
