# Step 3

Continuing with the `meta` elements, a `viewport` definition tells the browser how to render the page. Including one betters visual accessibility on mobile, and improves _SEO_ (search engine optimization).

Add a `viewport` definition with a `content` attribute detailing the `width` and `initial-scale` of the page.

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
--fcc-editable-region--
  <head>
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="styles.css" />
  </head>
--fcc-editable-region--
  <body>

  </body>
</html>
```

## Hints

1. You should create another `meta` element in the `head`.
2. You should give the `meta` a `name` attribute of `viewport`.
3. You should give the `meta` a `content` attribute of `width=device-width, initial-scale=1`.
