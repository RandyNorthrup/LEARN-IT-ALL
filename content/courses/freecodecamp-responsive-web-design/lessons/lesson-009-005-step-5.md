# Step 5

Below your `.header` element, create a new `div` element and assign it a `class` of `gallery`. This `div` will act as a container for the gallery images.

Inside that `.gallery` element, create nine `img` elements.

## Starter Code

```html
--fcc-editable-region--
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Photo Gallery</title>
    <link rel="stylesheet" href="./styles.css">
  </head>
  <body>
    <header class="header">
      <h1>css flexbox photo gallery</h1>
    </header>
  </body>
</html>
--fcc-editable-region--
```

## Hints

1. You should create a `div` element in your `body` element.
2. Your new `div` element should have a `class` with `gallery` set as the value.
3. Your new `div` element should come after your `.header` element.
4. Your `.gallery` element should have nine `img` elements.
