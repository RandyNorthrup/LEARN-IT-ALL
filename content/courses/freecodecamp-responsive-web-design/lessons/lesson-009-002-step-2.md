# Step 2

Within your `head` element, add a `meta` tag with the `name` set to `viewport` and the `content` set to `width=device-width, initial-scale=1`.

Also add a `meta` tag with the `charset` set to `UTF-8`.

## Starter Code

```html
--fcc-editable-region--
<!DOCTYPE html>
<html lang="en">
  <head>

  </head>
  <body>
  </body>
</html>
--fcc-editable-region--
```

## Hints

1. You should have two `meta` elements.
2. One `meta` element should have a `name` set to `viewport`, and `content` set to `width=device-width, initial-scale=1.0`.
3. Your other `meta` element should have the `charset` attribute set to `UTF-8`.
