# Reviewed course source

This directory is intentionally empty of course data.

A course may enter this directory only after its research dossier, prerequisite graph, activity matrix, original learner-facing instruction, assessment plan, and runtime design have passed their required reviews. Generated inventory and rejected legacy output do not belong here.

Course publication remains a separate explicit decision in `src/lib/data/publishedCourses.ts` after all repository release gates pass.
