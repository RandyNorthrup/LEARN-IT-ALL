## Overview

This directory contains hands-on project tutorials for building real-world applications from scratch. All projects are sourced from the "Build Your Own X" repository, which provides step-by-step guides for recreating favorite technologies.

### Philosophy

> "What I cannot create, I do not understand." — Richard Feynman

Projects teach through implementation. By building technologies from scratch, learners gain deep understanding of how systems work internally.

## Difficulty Levels

- **Beginner**: 0-6 months programming experience
- **Intermediate**: 6-18 months experience
- **Advanced**: 18+ months experience, familiar with systems programming

## How to Use Projects

### For Learners
1. Choose a project matching your skill level and interests
2. Follow the linked tutorial step-by-step
3. Implement the project in your preferred language
4. Experiment with extensions and improvements
5. Share your implementation in the community

### For Instructors
1. Assign projects as capstone exercises
2. Use projects to teach specific concepts
3. Create courses around project themes
4. Guide learners through implementations
5. Review and provide feedback on implementations

### Integration with Courses
Projects complement formal courses:
- **CompTIA Network+** → Network Stack, Web Server projects
- **Python OOP** → Game, Bot, Command-Line Tool projects
- **JavaScript** → Framework, Template Engine projects
- **Database course** → Database, Key-Value Store projects

## Contributing Projects

### Guidelines
1. **Quality**: Tutorial must be well-written and complete
2. **License**: Must be open source and freely available
3. **Language**: Clear instructions in English
4. **Completeness**: Code must compile/run successfully
5. **Attribution**: Proper credit to original authors

### How to Add
1. Find a tutorial matching our quality standards
2. Create entry in appropriate `projects.json`
3. Add tags and metadata
4. Submit pull request with documentation
5. Link to course if relevant
